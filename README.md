# Meu Portfólio Pessoal

Este é um projeto desenvolvido em **Angular** e publicado no GitHub Pages em:  
📍 [daniloercosta.github.io](https://daniloercosta.github.io)

## 📌 Sobre o Projeto

Este site é meu portfólio pessoal, criado para apresentar meus projetos, meu GitHub, e contar um pouco sobre minha trajetória profissional.  
O objetivo é compartilhar conhecimento, experiências e facilitar o contato para oportunidades e networking.

## 🧰 Tecnologias Utilizadas

- Angular
- TypeScript
- HTML & SCSS
- Angular Material
- GitHub Pages (para publicação)

## 🧩 Funcionalidades

- Página inicial com informações sobre mim
- Listagem dos projetos públicos do meu GitHub
- Exibição de imagens retiradas diretamente dos `README.md` dos repositórios
- Links para os repositórios e deploys (quando disponíveis)
- Layout responsivo e visual limpo

## 🚀 Como Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/daniloercosta/daniloercosta.github.io.git

# Acesse o diretório do projeto
cd daniloercosta.github.io

# Instale as dependências
npm install

# Execute o projeto
ng serve

# Acesse em: http://localhost:4200
📤 Deploy
